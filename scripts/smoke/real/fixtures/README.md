# Real-smoke fixtures

`signed-settle-fresh.json` — body for `POST /settle` with a real
ADR-036-signed authorization from the payer in `x402-cosmos/.env` to the
matching facilitator/recipient, on Cosmos testnet `grand-1`.

## Single-use, not committed

This file is generated by `00-prepare-fixtures.sh` on every smoke run.
Each authorization carries a fresh random 32-byte nonce and a tight
validity window (~55s from generation). After a successful `/settle` the
nonce is permanently consumed on chain; replays will fail with
`nonce_already_used`. Re-run `00-prepare-fixtures.sh` to regenerate.

The fixture is in `.gitignore` — do not commit a real signed payload.

## Regenerating manually

```bash
cd /home/govhub/suverse-pay
set -a && source /home/govhub/x402-cosmos/.env && set +a
/home/govhub/x402-cosmos/bin/fixture \
  --output scripts/smoke/real/fixtures/signed-settle-fresh.json
```

The generator reads payer mnemonic, recipient, facilitator grantee, and
network metadata from `x402-cosmos/.env`. See
`x402-cosmos/tools/fixture/main.go` for the full flag list.
